cin用法

发表于2023-12-12|更新于2023-12-12|C++
|字数总计:1.8k|阅读时长:7分钟|阅读量:

简介

C++中cin是标准输入流，用于从标准输入设备（通常是键盘）读取数据。cin的用法主要有以下几种：

1. cin >> 变量名

从标准输入流读取数据，并将其赋值给变量名。cin>>可以读取整数、浮点数、字符串等数据。

输出：

请输入整数：123 请输入浮点数：123.456 请输入字符：a 您输入的整数是：123 您输入的浮点数是：123.456 您输入的字符是：a

2. 一行输入多个

cin可以使用>>运算符在一行中输入多个变量。例如，cin >> a >> b; 表示从标准输入流读取两个数据，分别赋值给变量a和b。cin会自动根据变量的类型进行格式转换，也会忽略输入中的空白字符（空格，制表符，换行符）。

3. cin.get()方法

3.1 方法介绍🖇️

用于读取单个字符或一行字符串，返回值是读取到的字符的ASCII码值。

get方法有多种重载形式，可以指定读取的目标变量，读取的字符个数，以及读取的结束符。

例如，

  1. **cin.get(ch)**表示从标准输入流读取一个字符，赋值给变量ch。
  2. **cin.get(str, 10)**表示从标准输入流读取最多10个字符（不包括结束符），赋值给字符串变量str。
  3. **cin.get(str, 10, ‘.’)**表示从标准输入流读取最多10个字符，或者遇到’.’字符为止，赋值给字符串变量str。

4. cin.getline()用法

从标准输入流读取一行字符串。cin.getline()的第一个参数指定字符串变量，第二个参数指定分隔符。

getline方法主要有两种重载形式，可以指定读取的目标变量，读取的字符个数，以及读取的结束符。

  1. cin.getline(str, 10); 表示从标准输入流读取一行字符串，最多10个字符（不包括换行符），赋值给字符串变量str。

    请输入一行字符串：Hello, world! 您输入的字符串是：Hello, world!

  2. cin.getline(str, 10, ‘.’); 表示从标准输入流读取一行字符串，最多10个字符，或者遇到’.’字符为止，赋值给字符串变量str。

    请输入多行字符串，以#结束： This is a test for cin.getline()# 您输入的字符串是： This is a test for cin.getline()

6. cin()其他

在使用cin时，需要注意以下几点：

  • cin读取数据时，会将缓冲区中的所有数据读取完毕。如果缓冲区中没有数据，则cin将阻塞，直到有数据输入。
  • cin读取数据时，会忽略空格、制表符和换行符。
  • cin读取数据时，如果遇到非法数据，则cin将抛出异常。
  • cin可以使用peek方法来查看标准输入流中的下一个字符，但不从流中移除该字符。例如，ch = cin.peek(); 表示查看标准输入流中的下一个字符，赋值给变量ch，但该字符仍然在流中等待读取。
  • cin可以使用ignore方法来忽略标准输入流中的一个或多个字符，不进行读取。例如，cin.ignore(); 表示忽略标准输入流中的下一个字符。cin.ignore(10); 表示忽略标准输入流中的10个字符。cin.ignore(10, ‘.’); 表示忽略标准输入流中的10个字符，或者遇到’.’字符为止。
  • cin可以使用putback方法来将一个字符放回标准输入流中，以便下次读取。例如，cin.putback(ch); 表示将字符变量ch放回标准输入流中。
  • cin可以使用clear方法来清除标准输入流的错误状态，例如当读取失败或遇到文件结束符时。例如，cin.clear(); 表示清除标准输入流的错误状态，使其恢复正常。
  • cin可以使用eof方法来判断标准输入流是否已经到达文件结束符。例如，cin.eof(); 表示返回一个布尔值，如果标准输入流已经到达文件结束符，返回true，否则返回false。
  • cin可以使用good方法来判断标准输入流是否处于正常状态，没有发生错误或到达文件结束符。例如，cin.good(); 表示返回一个布尔值，如果标准输入流处于正常状态，返回true，否则返回false。
